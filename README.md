# Site Vitrine de Traduction

## Présentation

Ce projet consiste en un site vitrine développé sous WordPress pour <strong>HK Transaltion</strong>, une traductrice professionnelle. L'objectif de ce site est de présenter ses services de traduction, son expertise, et de permettre aux clients potentiels de la contacter facilement.

## Fonctionnalités

- **Présentation des Services** : Le site met en avant les différents services de traduction offerts, incluant les langues et les domaines de spécialisation.
  
- **Profil de l'Expertise** : Une section dédiée à la présentation du parcours professionnel et des qualifications des membres de l'équipe pour établir la crédibilité et la confiance auprès des clients.

- **Formulaire de Contact** : Les visiteurs peuvent facilement envoyer un message ou une demande de renseignement via un formulaire de contact, ce qui facilite la prise de contact.

- **Design Réactif** : Le site est conçu pour être accessible et facile à naviguer sur tous les appareils, y compris les ordinateurs de bureau, les tablettes et les smartphones.

## Technologies Utilisées

- **WordPress** : Système de gestion de contenu (CMS) utilisé pour développer le site.
- **Thème** : OceanWP.
- **Plugins Utilisés** :
  - UpdraftPlus : Sauvegarde et restauration automatique.
  - SEOPress : Optimisation SEO.
  - Royal Elementor Addons : Addons pour le constructeur de page Elementor.
  - Redirection : Gestion des redirections et des erreurs 404.
  - Imagify : Optimisation des images pour améliorer les performances.
  - Hustle : Collecte d'adresses e-mail avec des pop-ups.
  - Elementor : Constructeur de pages avec interface glisser-déposer.
  - Duplicate Page : Duplication facile des pages et publications.
  - Central Color Palette : Gestion centralisée de la palette de couleurs.
  - Broken Link Checker : Vérification des liens brisés sur le site.
  - WPForms Lite : Création de formulaires par glisser-déposer.
